# Toolkit for tomography reconstruction

A python-based GUI package developed for tomography recontruction.
Optimized for data taken at the FXI beamline at NSLS-II at Brookhaven National Laboratory.

ML denoisoing on tomo reconstruction is included for test

## Installation

Note: need to intall pytorch with cuda enabled
```
git clone https://github.com/gmysage/FXI_tomo_util.git


```
